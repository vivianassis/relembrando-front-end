Tudo isso se deve as chamadas "TAGS" do HTML.

Como Formatar o HTML - Parte 3

Através da tag 'input', podemos atribuir diversas funções a campos dentro de nossas páginas.
Essa tag e seus atributos são muito usados para criar e formatar formulários web.
Alguns exemplos de formatação para a tag 'input' estão descritos na lista a seguir:

  • Para criar um Campo 'texto', basta usar o atributo 'type' com o valor 'text' no input:
    < input type="text" / > =
  • Para o Campo 'cor', o atributo type recebe o valor 'color':
    < input type="color" / > =
  • Criamos o Campo 'número' usando o valor 'number' no type do input:
    < input type="number" / > =

Seguindo essa mesma lógica é fácil conseguir visualizar e criar formulários super legais.

Como Formatar o HTML - Parte 4

Listas ordenadas (TAG 'ol')

Criadas para descrever uma sequência determinada de itens:
Listas ordenadas são identificadas por possuírem uma ordem de grandeza
Exemplos: (1-2-3) (I - II - III)

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5
  1. Item 1
    1. Item 1.1
    2. Item 1.2
    3. Item 1.3
    4. Item 1.4
    5. Item 1.5
Listas não ordenadas (TAG 'ul')

Criadas para descrever uma sequência de itens onde a ordem não importa:
A lista não ordenada se vale de marcadores para sua formatação.

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5
    • Item 5.1
    • Item 5.2
    • Item 5.3
    • Item 5.4

Como Formatar o HTML - Parte 5

Imagens no HTML tem uma tag específica, a tag 'img'. (< img src="" alt="" >)
Existem com ela como podemos ver no exemplo acima, dois atributos essenciais, que são o 'src' (que determina o local de onde o navegador buscará a imagem) e o 'alt', que apesar de opcional, é bastante importante, pois ele da uma breve descrição da imagem, facilitando sua leitura por programas de acessibilidade.
Existem também, vários atributos não obrigatórios, para melhor se adequar a cada página como, por exemplo, a tag WIDTH:


Width = usada para dar um tamanho a imagem

Como Formatar o HTML - Parte 6

Como sabemos, uma das funcionalidades mais usadas nas páginas Web são os Hiperlinks, ou simplesmente LINKS, eles são responsáveis pela navegabilidade na internet, e são muito importantes para a experiência do usuário ser cada vez mais completa. Para inserir LINK, usamos a TAG 'a', onde o atributo href recebe o valor do link e precisamos colocar um conteúdo visível, entre as tags.

A tag 'a':

< a href="www.meusite.com" >< h1 >Meu site aqui< /h1 >< /a >

Podemos usar o atributo target para definir como o site fará a navegabilidade até o link informado:

'_blank' como valor, para que o link abra em uma nova aba

DIO.me

'_self' como valor, para que o link abra na mesma guia

DIO.me, mas abrindo na mesma aba

Podemos também, usar o atributo 'title' na tag 'a'. Ele serve para mostrar um tooltip sobre o link
